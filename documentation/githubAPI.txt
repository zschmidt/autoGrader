Must authenticate with API to go above 60 requests per hour


curl -i -u [your_username] https://api.github.com



Owner of the app must register the application:
https://github.com/settings/developers





Adding Scopes:
Simply add them to the validation page -- see here:
https://developer.github.com/v3/oauth/#scopes




After validating the app, need to post back to github:
https://developer.github.com/guides/basics-of-authentication/#providing-a-callback



Client Secret:
-This is an environment variable on the server (client_secret)
